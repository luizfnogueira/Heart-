Este diretório contém os recursos do jogo Heart.

Para o jogo funcionar corretamente, você precisa adicionar os seguintes arquivos:

1. background_music.mp3 - Música de fundo para o jogo

Você pode usar qualquer música de fundo de sua preferência, desde que seja compatível com o formato MP3 e com a biblioteca Raylib.

Lembre-se de verificar os direitos autorais das músicas que você utilizar.
